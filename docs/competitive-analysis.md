# YumeTree 名称整理メモ

作成日: 2026-05-16
ステータス: Phase 1 公開表記変更の判断メモ

## 結論

公開表示名は `YumeTree`、日本語表記は `ユメツリー` に統一する。

旧名称の `ユメログ` / `YumeLog` は、夢を記録するアプリとして意味は伝わりやすい一方、類似した名称との区別がつきにくい可能性がある。公開面での識別性を高めるため、ブランド名を変更する。

## 公式表記

- ブランド名: `YumeTree`
- 日本語表記: `ユメツリー`
- サブコピー: `モルペウスと育てるAI夢ノート`

`YumeTree` 単体では何のアプリか伝わりにくいため、LP、OGP、README など初見ユーザーが見る場所では、必ず `AI夢ノート`、`モルペウス`、`夢を記録` などの説明を併記する。

## 名称の混同リスク

- `ユメログ` / `YumeLog` 系の名称は、夢日記・夢記録アプリの説明語に近く、サービス固有の名前としては埋もれやすい。
- 類似した名称が複数ある場合、初めて見るユーザーが別サービスと誤認する可能性がある。
- `YumeTree` は造語として識別しやすく、夢の記録が積み重なって育つ体験を表現しやすい。

## Phase 1 の変更範囲

今回変更するもの:

- LP、ヘッダー、フッターなどの公開表示名
- metadata title / description / OGP / Twitter card
- PWA manifest の `name` / `short_name`
- README のアプリ名と説明
- 現行ブランドを説明する docs

今回変更しないもの:

- DB名
- backend内部名
- GitHubリポジトリ名
- Stripe商品名
- 環境変数名

- 使わない: `Yume Tree`
